# 🎵 Database Practice — Songs & Artists

Welcome to my SQL playground 🚀
This repository contains a **mini music database project** that I built purely for practicing and mastering SQL queries.

---

## 📚 Project Overview

This database simulates a simple music system with two core tables:

* 🎶 **Songs** — stores song information
* 🎤 **Artists** — stores artist details

The goal of this project was to:

* Practice SQL syntax
* Improve query writing speed ⚡
* Test joins, filters, and aggregations
* Experiment with real-world database logic

---

## 🧠 Skills Practiced

✨ SELECT queries
🔗 JOIN operations
📊 Aggregations (COUNT, AVG, etc.)
🎯 Filtering & conditions
📑 Sorting & grouping

---

## 🗂 Database Structure

```
songs
 ├─ id
 ├─ title
 ├─ artist_id
 └─ duration

artists
 ├─ id
 ├─ name
 └─ country
```
